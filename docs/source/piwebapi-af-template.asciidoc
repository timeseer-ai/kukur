== PI Asset Framework Element Template using Web API

Sources of type `piwebapi-af-template` provide the same capabilities as sources of the `piwebapi-af` type.

Defaults differ for compatibility reasons and are documented below:

```toml
[source.<name>]
type = "piwebapi-af-template"
attributes_as_fields = true # set to false to use the attribute name as the "series name" tag.
# all other options of the piwebapi-af source are supported here.
```
